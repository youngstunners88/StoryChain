
**Blue Ocean Scoring Algorithm:**
- Low competition + High growth = Blue Ocean (score 7-10)
- Pain signals multiply potential value
- Engagement metrics boost confidence
- Automatic action hints (FAST MVP or PIVOT)

**Pain Signal Detection:**
- `automation_need` - Users seeking automation
- `complexity_pain` - Tools too complex
- `price_sensitivity` - Overpriced alternatives
- `tool_gap` - Missing tools in ecosystem
- `explicit_need` - Direct requests for solutions
- `knowledge_gap` - Learning opportunities

**New API Endpoints:**
- `GET /scan` - Trigger manual opportunity scan
- `GET /opportunities?min_score=7&source=github` - List discovered opportunities
- `PATCH /opportunity/:id` - Update status (validated, pursuing, monetized)

**Automatic Behavior:**
- Scans all sources every 6 hours
- Stores opportunities in memory
- Flags high-value opportunities (Blue Ocean >= 7)
- Integrates with Agent Lightning for learning
