# TinyFish Web Agent

Autonomous web automation for revenue-generating tasks.

## Usage

Use the `tinyfish` script to run web automations.

## Capabilities

- Extract data from dynamic websites
- Monitor prices and availability
- Automate form submissions
- Handle authentication
- Execute parallel operations
- Generate structured outputs
