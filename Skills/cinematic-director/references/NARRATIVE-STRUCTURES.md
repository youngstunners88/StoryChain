# Narrative Structures Reference

A comprehensive guide to storytelling structures for cinematic video content.

## The Three-Act Structure

### Act 1: Setup (10-20% of content)

**Purpose:** Establish world, introduce characters, set tone

**Key Elements:**
- World establishment (where are we?)
- Character introduction (who is this?)
- Tone setting (how should we feel?)
- Inciting incident (what disrupts the status quo?)

**Visual Approach:**
- Wider shots for context
- Steady, controlled movements
- Clear, legible compositions

**Shot Distribution:**
- 2-4 establishing shots
- 1-2 character introduction shots
- 1 inciting incident shot

### Act 2: Confrontation (60-70% of content)

**Purpose:** Present challenges, build tension, show struggle

**Key Elements:**
- Rising action (escalating stakes)
- Obstacles (what stands in the way?)
- Character development (how do they change?)
- Midpoint twist (what shifts perspective?)

**Visual Approach:**
- Closer shots for intimacy
- More dynamic movements
- Increased tension in compositions

**Shot Distribution:**
- Multiple challenge/obstacle shots
- Character reaction shots
- Building momentum shots

### Act 3: Resolution (15-20% of content)

**Purpose:** Climactic moment, emotional release, new equilibrium

**Key Elements:**
- Climax (the decisive moment)
- Resolution (how is conflict resolved?)
- New normal (what has changed?)

**Visual Approach:**
- Powerful compositions
- Emotionally-charged movements
- Clear visual resolution

**Shot Distribution:**
- 1-2 climax shots
- 1-2 resolution shots
- 1 final image

## The Hero's Journey (Condensed)

### 12-Stage Framework

1. **Ordinary World** - Status quo, normal life
2. **Call to Adventure** - Disruption, opportunity
3. **Refusal of Call** - Hesitation, fear
4. **Meeting Mentor** - Guidance, preparation
5. **Crossing Threshold** - Entering unknown
6. **Tests, Allies, Enemies** - Building competence
7. **Approach to Inmost Cave** - Preparing for ordeal
8. **Ordeal** - Major confrontation
9. **Reward** - Gaining something significant
10. **Road Back** - Return journey
11. **Resurrection** - Final test
12. **Return with Elixir** - Transformed state

### Condensed for Short Form (6 stages)

```
1. ORDINARY WORLD → Show normal state
2. CALL + CROSSING → Disruption and commitment
3. TESTS + APPROACH → Building toward climax
4. ORDEAL → Major confrontation
5. REWARD + ROAD BACK → Victory and return
6. RESURRECTION + RETURN → Transformed state
```

## The Kishōtenketsu Structure

**East Asian narrative structure** - Four acts without conflict:

```
ACT 1 (Introduction): Establish elements
ACT 2 (Development): Elaborate on elements  
ACT 3 (Twist): Unexpected turn, new perspective
ACT 4 (Conclusion): Resolution with new understanding
```

**Visual Approach:**
- ACT 1: Clear, simple compositions
- ACT 2: Developing complexity
- ACT 3: Unexpected angle/perspective
- ACT 4: Harmonious resolution

**Use for:** Atmospheric pieces, mood videos, artistic content

## The Fichtean Curve

**Series of crises building to climax:**

```
START → Crisis 1 → Crisis 2 → Crisis 3 → CLIMAX → Resolution
```

**Visual Approach:**
- Each crisis has visual escalation
- Building intensity through shots
- Climax is most dynamic
- Resolution provides release

**Use for:** Action sequences, tension-building content

## The In Media Res Structure

**Starting in the middle of action:**

```
HOOK (mid-action) → FLASHBACK (context) → PRESENT (resolution)
```

**Visual Approach:**
- HOOK: Dynamic, confusing, intriguing
- FLASHBACK: Clearer, explanatory
- PRESENT: Resolution with full context

**Use for:** Engaging opens, mystery reveals

## The Circular Structure

**End connects back to beginning:**

```
A → B → C → D → A (transformed)
```

**Visual Approach:**
- Begin and end with similar compositions
- Show transformation through differences
- Same location, different feeling

**Use for:** Character arcs, transformation stories

## Emotional Arcs

### The 6 Basic Emotional Arcs

Based on Kurt Vonnegut's story shapes:

1. **Rags to Riches** (Rise)
   - Start: Low point
   - End: High point
   - Visual: Dark to light, low to high angles

2. **Riches to Rags** (Fall)
   - Start: High point
   - End: Low point
   - Visual: Light to dark, high to low angles

3. **Man in a Hole** (Fall-Rise)
   - Start: Good
   - Middle: Bad (hole)
   - End: Better
   - Visual: Good → Dark → Light

4. **Icarus** (Rise-Fall)
   - Start: Bad
   - Middle: Good (peak)
   - End: Bad
   - Visual: Dark → Light → Dark

5. **Cinderella** (Rise-Fall-Rise)
   - Start: Bad
   - Middle 1: Good (ball)
   - Middle 2: Bad (midnight)
   - End: Good (transformation)
   - Visual: Dark → Light → Dark → Bright

6. **Oedipus** (Fall-Rise-Fall)
   - Start: Good
   - Middle 1: Bad (truth)
   - Middle 2: Good (resolution)
   - End: Tragic acceptance
   - Visual: Light → Dark → Light → Shadow

## Scene Transitions

### Types of Transitions

1. **Cut** - Direct transition
   - Use for: Continuity, pace, action
   - Visual: Match action, eyeline, or emotion

2. **Fade** - Gradual transition
   - Fade out: End of scene
   - Fade in: New scene/time
   - Use for: Time passage, emotional beats

3. **Dissolve** - Overlapping images
   - Use for: Time passage, memories, dreams
   - Visual: Two images blend

4. **Wipe** - Directional transition
   - Use for: Style, energy, genre films
   - Visual: One image pushes another

5. **Match Cut** - Visual continuity
   - Use for: Connection, contrast
   - Visual: Similar shape/color/movement

6. **Jump Cut** - Time compression
   - Use for: Energy, discontinuity, style
   - Visual: Same subject, different time

### Match Cut Types

| Type | Match Element | Example |
|------|---------------|---------|
| Action | Movement | Door opening → Eye opening |
| Graphic | Shape | Circle → Circle |
| Colour | Colour | Red dress → Red sky |
| Sound | Audio | Glass break → Bell ring |
| Emotion | Expression | Anger → Anger |
| Eyeline | Look direction | Character looks → What they see |

## Scene Beats

### The Beat Sheet Method

Break scenes into emotional beats:

```
BEAT 1: Enter [emotion]
BEAT 2: Complication [emotion]
BEAT 3: Reaction [emotion]
BEAT 4: Decision [emotion]
BEAT 5: Action [emotion]
```

**Example:**
```
BEAT 1: Enter room - CURIOSITY (medium shot, push in)
BEAT 2: Notice danger - TENSION (close-up, static)
BEAT 3: Fear reaction - FEAR (extreme close-up, slight shake)
BEAT 4: Decide to run - DETERMINATION (medium, tilt up)
BEAT 5: Sprint to exit - URGENCY (tracking shot, handheld)
```

## Pacing Guide

### Shot Duration by Type

| Shot Type | Typical Duration | Effect |
|-----------|------------------|--------|
| Extreme wide | 4-8 seconds | Establishing, scale |
| Wide | 3-5 seconds | Environment |
| Full body | 3-4 seconds | Character presence |
| Medium | 2-4 seconds | Interaction |
| Close-up | 2-3 seconds | Emotion |
| Extreme close-up | 1-2 seconds | Impact, detail |

### Pacing by Genre

| Genre | Avg Shot Duration | Character |
|-------|-------------------|-----------|
| Action | 1-3 seconds | Fast, urgent |
| Drama | 4-8 seconds | Contemplative |
| Horror | 3-10 seconds | Building tension |
| Comedy | 2-4 seconds | Quick timing |
| Documentary | 5-15 seconds | Informative |
| Music Video | 1-2 seconds | Rhythmic |

## Pro Tips

1. **Every shot serves the story** - No filler
2. **Emotion over plot** - Feel first, understand second
3. **Less is more** - Trust the audience
4. **Contrast creates interest** - Vary shots, angles, pace
5. **End strong** - Final image lingers longest
