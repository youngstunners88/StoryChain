# CreativeForge Skill

CreativeForge is your all-in-one creative studio for generating marketing assets, visual content, and multimedia for your projects.

## Capabilities

### Image Generation
- Generate AI images from text prompts using `generate_image`
- Edit existing images with `edit_image`
- Search for reference images with `image_search`

### Video Generation
- Create cinematic videos from images using `generate_video`
- Multi-scene video production for storytelling
- Character-consistent video generation

### Diagrams & Visualizations
- Create D2 diagrams with `generate_d2_diagram`
- System architecture visualization
- Flowcharts and process diagrams

### Content Creation
- Marketing copy and ad copy
- Social media content
- Product descriptions and launch materials

## Usage

| Task | Command/Tool |
|------|--------------|
| Generate image | `generate_image` tool |
| Edit image | `edit_image` tool |
| Create video | `generate_video` tool |
| Make diagram | `generate_d2_diagram` tool |
| Find references | `image_search` tool |

## Integration Points

- **iHhashi**: Product images, restaurant photos, marketing materials
- **Boober**: Vehicle imagery, driver visuals
- **TikTok Marketing**: Post images, slideshow content
- **Landing Pages**: Hero images, illustrations

## Best Practices

1. **Be specific** - Detailed prompts produce better results
2. **Reference existing** - Use image_search to find style references
3. **Iterate** - Start simple, refine based on results
4. **Match brand** - Maintain consistent visual style across outputs
