# Recipe: scaffold-next-app

## Description
Create a Next.js 14 application with App Router, Tailwind CSS, and essential integrations.

## Prerequisites
- Node.js 18+
- Bun or npm
- Git

## Steps
1. Create Next.js app with TypeScript
2. Configure Tailwind CSS with custom theme
3. Set up shadcn/ui components
4. Add Lucide React icons
5. Create base layout and navigation
6. Set up environment variables
7. Initialize git repository

## Files Created
- package.json
- tsconfig.json
- tailwind.config.ts
- app/layout.tsx
- app/page.tsx
- app/globals.css
- lib/utils.ts
- .env.example
- .gitignore

## Variables
- PROJECT_NAME
- APP_TITLE
