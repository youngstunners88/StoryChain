## Required Skills

### Core Skills
1. **LLM Fundamentals**
   - Understanding temperature, tokens, context windows
   - Model-specific quirks (GPT-4, Claude, Gemini, Llama)
   - Embeddings and retrieval-augmented generation (RAG)

2. **Prompt Patterns**
   - Chain-of-thought prompting
   - Few-shot learning
   - System prompt architecture
   - Multi-step reasoning chains

3. **Evaluation Methods**
   - A/B testing prompts
   - Performance metrics
   - Iteration strategies

### Advanced Skills (Higher Rates)
- LangChain and agentic frameworks
- Fine-tuning concepts
- Python for prompt automation
- Integration with APIs and workflows

## Action Steps to Pursue

### Phase 1: Skill Building (Week 1-2)

1. **Learn Fundamentals**
   - Take free prompt engineering courses
   - Practice with ChatGPT/Claude/Gemini
   - Build prompt library for common use cases

2. **Document Your Work**
   - Create portfolio of 10+ example prompts
   - Include before/after examples
   - Show measurable improvements

3. **Understand Business Applications**
   - Customer service automation
   - Content generation
   - Data extraction and analysis
   - Code generation assistance

### Phase 2: Portfolio & Presence (Week 3-4)

1. **Create Portfolio**
   - Case studies with measurable results
   - Example prompts for different industries
   - Video demonstrations of prompt systems

2. **Set Up Profiles**
   - Upwork profile optimized for "AI prompt engineer"
   - PeoplePerHour profile
   - LinkedIn profile highlighting AI skills
   - PromptBase seller account

3. **Initial Pricing Strategy**
   - Start at $40-60/hr to build reviews
   - Offer competitive project rates
   - Consider 20% discount for first 5 clients

### Phase 3: Client Acquisition (Week 5+)

1. **Cold Outreach**
   - Target AI startups and agencies
   - LinkedIn direct messages
   - 20+ outreach per day

2. **Platform Strategy**
   - Apply to 10+ relevant jobs daily on Upwork
   - Submit proposals within 1 hour of posting
   - Focus on smaller jobs initially for reviews

3. **Content Marketing**
   - Write LinkedIn posts about prompt engineering
   - Share prompt examples and tips
   - Build authority in the space

### Phase 4: Scale (Month 3+)

1. **Raise Rates**
   - Increase rates 25-50% after 10+ successful projects
   - Target $80-150/hr for ongoing work

2. **Build Systems**
   - Templates for common prompt patterns
   - Reusable evaluation frameworks
   - Automated testing scripts

3. **Consider Team**
   - Hire junior prompt engineers
   - Train and delegate
   - Move to consultant/manager role
