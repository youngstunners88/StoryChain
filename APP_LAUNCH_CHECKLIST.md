# APP LAUNCH CHECKLIST

Source: [Abhishek Dutta (@abhishek_ux)](https://www.instagram.com/reel/DUdklo9AoqO/)

---

## App Store Stuff

- [ ] App title with keywords
- [ ] Subtitle (don't leave empty)
- [ ] App description (hook in first 2 lines)
- [ ] Keywords researched and added
- [ ] Screenshots (show benefits, not features)
- [ ] App preview video (optional but helps)
- [ ] Privacy policy URL
- [ ] Support URL
- [ ] App category selected
- [ ] Age rating completed

---

## Website Stuff

- [ ] Landing page live
- [ ] Open Graph tags set (og:title, og:description, og:image, og:url)
- [ ] Favicon added
- [ ] Mobile responsive
- [ ] SSL certificate active (https)
- [ ] Download/CTA button working

---

## SEO Stuff

- [ ] Google Search Console connected
- [ ] Bing Webmaster Tools connected
- [ ] Sitemap submitted
- [ ] IndexNow configured
- [ ] Meta title and description set
- [ ] Robots.txt file in place

---

## Marketing Stuff

- [ ] Launch post drafted
- [ ] Social media assets ready (images, captions)
- [ ] Email list notified
- [ ] Product Hunt listing prepped (if using)
- [ ] Friends/community ready to support

---

## Legal Stuff

- [ ] Privacy policy written and linked
- [ ] Terms of service written and linked
- [ ] Data handling documented
- [ ] GDPR compliance (if applicable)
- [ ] Cookie notice (if website tracks)

---

> Save this. Use it. Don't launch without it.
