# AI Business Workspace — AI Prompts Cheat Sheet

Quick-reference for all AI prompts in this workspace. Copy, paste, and fill in the brackets.

---

## 📋 PROJECTS & TASKS

### Task Breakdown

```
Break down this project into actionable subtasks:

PROJECT: [Insert project name/description]
CONTEXT: [Business type, team size, deadline, constraints]

Provide 5-15 specific tasks with priority (Critical/High/Medium/Low), estimated effort (Quick/Medium/Large), dependencies, and helpful notes.
```

### Weekly Planning

```
Help me plan my week:

TASKS: [Paste task list with priorities and due dates]
AVAILABILITY: [Hours per day]
CONSTRAINTS: [Meetings, commitments]

Suggest: tasks to defer, optimal scheduling, risks, and daily schedule.
```

## 📅 CONTENT CALENDAR

### Generate Content Ideas

```
Generate 10 content ideas:

BUSINESS: [Type]
AUDIENCE: [Target audience]
PLATFORM: [Instagram/LinkedIn/Twitter/TikTok/YouTube]
GOAL: [Awareness/Engagement/Leads/Sales]

For each: title, content type, hook, outline, CTA, and why it works.
```

### Write Educational Post

```
Write a [platform] educational post:

TOPIC: [Topic]
KEY POINTS: [3 bullet points]
STYLE: Short paragraphs, personal insight, engaging question
LENGTH: [50-100/100-200/200-300 words]
Include 3-5 hashtags.
```

### Write Behind-the-Scenes Post

```
Write a [platform] behind-the-scenes post:

SHOWING: [What you're showing]
STORY ANGLE: [What makes it interesting]
LESSON: [What audience should learn]
LENGTH: [word count]
Make it authentic and relatable.
```

### Write Promotional Post

```
Write a non-salesy promotional post:

PRODUCT: [Name and description]
BENEFIT: [Primary value]
SOCIAL PROOF: [Testimonial or result]
OFFER: [Discount or bonus]
CTA: [What should they do]
Lead with value, focus on transformation, create urgency. [word count]
```

## 👥 CUSTOMER FOLLOW-UPS

### Re-engage Cold Lead

```
Write a re-engagement email:

LEAD: [Name]
BUSINESS: [Their company]
LAST INTERACTION: [When/what discussed]
PAIN POINTS: [Their problems]
MY OFFER: [What I sell]
RECENT UPDATE: [New feature/content]

Casual, provide value, soft CTA, under 150 words.
```

### Post-Purchase Check-In

```
Write a post-purchase follow-up:

CUSTOMER: [Name]
PRODUCT: [What they bought]
PURCHASE DATE: [Date]
THEIR GOALS: [What outcome they wanted]

Thank them, ask how it's going, offer help, request feedback. Under 150 words.
```

### At-Risk Customer Retention

```
Write a retention email:

CUSTOMER: [Name]
PRODUCT: [What they use]
SIGNS OF CHURN: [Low usage, missed payment, etc.]
ORIGINAL GOALS: [What they wanted to achieve]

Acknowledge, offer solutions, provide incentive, ask what would help. Empathetic, under 200 words.
```

### VIP Appreciation

```
Write a VIP customer appreciation email:

CUSTOMER: [Name]
TIME AS CUSTOMER: [Duration]
TOTAL VALUE: [Amount]
THEIR SUCCESS: [Results achieved]

Specific gratitude, acknowledge success, offer exclusive benefit. Warm, personal, under 150 words.
```

## 📚 RESEARCH

### Summarise Content

```
Summarise this content:

CONTENT: [Paste article/describe video/book]

Provide:
- One-sentence summary
- 3-5 key takeaways
- Actionable ideas
- Questions it raises
- Related topics
- Notable quotes
```

### Competitor Analysis

```
Analyse this competitor:

COMPETITOR: [Company name/website]
MY BUSINESS: [Your description]
FOCUS ON: [Pricing/positioning/features/marketing/strengths]

Analyse: positioning, offering, marketing, strengths, weaknesses, opportunities for me, threat level.
```

## 📝 MEETING NOTES

### Pre-Meeting Prep

```
Prepare for this meeting:

TYPE: [Sales/client/kickoff/review/etc.]
ATTENDEES: [Names and roles]
MY GOAL: [What I want]
THEIR GOALS: [What they probably want]
CONTEXT: [Background]

Provide: objectives, agenda with timing, questions to ask, potential objections with responses, talking points, don't forget items.
```

### Post-Meeting Summary

```
Summarise this meeting:

MEETING: [Title, date, attendees]
RAW NOTES: [Paste messy notes]

Provide:
- 2-3 sentence executive summary
- Decisions made
- Action items with owners and due dates
- Next steps
- Blockers/risks
- Key insights
- Open questions
```

## 💰 FINANCIAL INSIGHTS

### Monthly Financial Analysis

```
Analyse my finances:

BUSINESS: [Type]
PERIOD: [Month/Quarter]
REVENUE: [Breakdown by source]
EXPENSES: [By category]
METRICS: [Total revenue, expenses, profit, margin, growth]

Provide: health score (1-10), revenue analysis, expense analysis, profitability trends, alerts, 3 recommendations, next month forecast.
```

### Pricing Strategy

```
Help me set pricing:

PRODUCT: [Description]
TARGET MARKET: [Who buys]
CURRENT PRICE: $[X]
COMPETITORS: [Their prices]
MY COSTS: [Delivery cost]
VALUE: [Customer outcome]

Provide: 3 pricing options with margins and pros/cons, recommended price, pricing psychology tactics, validation experiment.
```

## 💡 PRO TIPS

1. **Add context freely** — More details = better output
2. **Chain prompts** — Use output from one prompt as input to another
3. **Iterate** — Ask AI to revise: "Make it shorter" or "Add more urgency"
4. **Save winners** — When a prompt works well, note what made it effective
5. **Customise for your voice** — Add style guidelines that match your brand
