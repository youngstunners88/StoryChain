## Project Description
Boober Missing is a comprehensive ride-hailing platform designed for the South African market. 
It connects passengers with verified drivers through a mobile app, featuring real-time GPS tracking, 
transparent pricing, and a focus on safety.

## Core Features
- User registration and authentication (email, phone, social)
- Driver onboarding and verification
- Real-time ride booking and dispatch
- GPS tracking and route optimization
- Dynamic fare calculation
- In-app payments and digital wallet
- Rating and review system
- Admin dashboard for operations
- Driver earnings and analytics
- Safety features (SOS, ride sharing)

## Technology Stack
- Backend: Node.js, Express, TypeScript, Prisma, PostgreSQL
- Mobile: React Native, Expo
- Web: Next.js, React
- Real-time: WebSocket, Socket.io
- Payments: PayStack, Stripe
- Maps: Google Maps API, Mapbox
- Storage: AWS S3
- Authentication: JWT, OAuth2
