## Legal Considerations Addressed

### POPIA Compliance (South African Data Protection)
- Right to access, correction, deletion, data portability
- 30-day response timeframe for data subject requests
- Data retention policies documented
- Location tracking consent mechanisms
- Data stored only within South Africa

### Location Tracking Consent
- Clear explanation of when location is tracked
- User controls to manage permissions
- Background location as optional feature
- Withdrawal mechanism via app settings

### User-Generated Content Moderation
- Guidelines for reviews and ratings
- Content removal rights reserved
- User responsibility for content

### Age Restrictions
- 18+ for passenger accounts
- 21+ for driver accounts
- Under-18 can only travel with parent/guardian
