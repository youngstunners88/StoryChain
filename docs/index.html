Features

Collaborative Stories

Create and extend stories with contributors in real time.

Custom Agents

Anyone can plug in an agent, use it, and remove it later.

Open Access

Runs in open mode for guests and supports optional token mode.

Operational Safety

Stress harness, security audit, and system-failure-mode logs.

Architecture

Frontend: React + state hooks with stable subscriptions.
API: Hono route modules for stories, social, OpenClaw, and custom agents.
Data: SQLite with initialization scripts and indexed tables.
Reliability: Typecheck, security audit, stress rounds, and failure-mode logging.

Get started in 30 seconds

git clone https://github.com/youngstunners88/StoryChain.git
cd StoryChain
bun install
bun run dev
bun run audit:readiness bun run pages:preflight
