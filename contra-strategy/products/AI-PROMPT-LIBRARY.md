## BONUS: PROMPT ENGINEERING TIPS

### Technique 1: Chain of Thought
```
Before answering, think through this step by step:
1. What is the core question?
2. What context is needed?
3. What are the possible approaches?
4. Which approach is best?
5. How do I implement it?
```

### Technique 2: Few-Shot Learning
```
Here are 3 examples of [OUTPUT TYPE]:
Example 1: [EXAMPLE]
Example 2: [EXAMPLE]
Example 3: [EXAMPLE]
Now create a similar output for: [NEW INPUT]
```

Act as [ROLE] with the following characteristics:

- Experience: [YEARS]
- Perspective: [ANGLE]
Now answer: [QUESTION]
